# Sudoku Game 
 
Juego de Sudoku de escritorio en **Java + JavaFX**, con tablero de 6x6 generado automáticamente.
 
## Características
 
- Generación automática de tableros válidos.
- Validación en tiempo real (fila, columna, subcuadro).
- Contador de errores (máximo 3).
- Sistema de pistas.
- Cronómetro de partida.
## Tecnologías
 
- Java 17 · JavaFX 17 · Maven · JUnit 5
